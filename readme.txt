To run the app you have to have nodejs installed. If you do not have nodejs installed go to 
https://nodejs.org/en/download/ and install node for your OS

Once you have Node installed go you your terminal(console) go to this project root directory and type "npm install".
This will install all the node modules required for this application.

Make sure you have gurnt client installed: In your terminal type "npm install -g grunt-cli"


In your terminal go to the project directory and type "grunt test".
